# Instituto Superior de Engenharia de Coimbra - Programação Avançada 2019/2020
Este trabalho prático foi realizado no âmbito da unidade curricular de Programação Avançada com o objetivo de absorver conhecimento de programação orientada a objetos em Java bem como outras funcionalidades dentro desta linguagem de programação.

## Objetivos

1. Definição e implementação do modelo de dados e da máquina de estados que representa o jogo de acordo com os padrões dados nas aulas como, por exemplo, estados polimórficos e fábrica de objetos.
2. Implementação completa e funcional do jogo 'Planet Bound' com interface de utilizador em modo gráfico (GUI).
